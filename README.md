# OS Cache Proxy

Многопоточный кэширующий HTTP-прокси сервер.

## Сборка

```bash
make
```

Для очистки:
```bash
make clean
```

## Запуск

```bash
./proxy [OPTIONS]
```

### Опции

- `-p, --port PORT` - порт для прослушивания (по умолчанию: 8080)
- `-c, --cache SIZE` - размер кэша в МБ (по умолчанию: 100)
- `-h, --help` - показать справку

### Примеры

```bash
./proxy -p 8080 -c 200
./proxy --port 3128 --cache 500
```

## Использование

Настройте браузер на использование HTTP прокси:
- Host: localhost
- Port: 8080 (или указанный при запуске)

## Остановка

Для корректного завершения используйте `Ctrl+C` или отправьте `SIGTERM`:
```bash
kill -TERM <pid>
```
